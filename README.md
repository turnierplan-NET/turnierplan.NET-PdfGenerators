# turnierplan.NET | PDF Generators

*Additional document generators which aren't part of the main application*

## Introduction

This repository contains a .NET console application that can generate some various document types for entire folders containing multiple tournaments. At some point, these documents might be integrated into **turnierplan.NET** natively. See the [issue in the main repo](https://github.com/turnierplan-NET/turnierplan.NET/issues/301) for more information.
